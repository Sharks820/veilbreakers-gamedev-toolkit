  <action>
    Create `Tools/mcp-toolkit/src/veilbreakers_mcp/shared/texture_ops.py` with these functions:

    **1. generate_uv_mask(polygons, texture_size, feather_radius=5) -> bytes** (TEX-02)
    - Create L-mode (grayscale) image `texture_size x texture_size`, filled with 0 (black)
    - Use `ImageDraw.Draw(mask)` to fill each polygon with 255 (white)
    - Convert UV coords (0-1) to pixel coords: `(int(u * texture_size), int((1 - v) * texture_size))`
    - CRITICAL SEAMLESS BLENDING: Apply Gaussian blur to create feathered edges:
      ```python
      from PIL import ImageFilter
      # Create sharp mask first
      sharp_mask = mask.copy()
      # Blur for feathering
      blurred = mask.filter(ImageFilter.GaussianBlur(radius=feather_radius))
      # Composite: use sharp mask interior with blurred edges
      # Result: 255 in solid interior, gradient 0-255 at edges
      mask = Image.composite(sharp_mask, blurred, sharp_mask)
      ```
      Alternative approach: erode the sharp mask by feather_radius, then blur the original mask, then composite the eroded interior (full 255) with the blurred outer edge. This gives a flat interior with soft falloff.
    - Return PNG bytes via `io.BytesIO`
    - Also return the PIL Image object for chaining: `generate_uv_mask_image(...)` variant

    **2. apply_hsv_adjustment(image_bytes, mask_bytes, hue_shift=0.0, saturation_scale=1.0, value_scale=1.0) -> bytes** (TEX-04)
    - Load image and mask from bytes
    - Convert image to RGB if not already
    - For each pixel where mask > 0:
      - Convert RGB to HSV via `colorsys.rgb_to_hsv(r/255, g/255, b/255)`
      - Apply: hue = (hue + hue_shift) % 1.0, sat = clamp(sat * saturation_scale, 0, 1), val = clamp(val * value_scale, 0, 1)
      - Convert back: `colorsys.hsv_to_rgb(h, s, v)` then scale to 0-255
    - CRITICAL SEAMLESS BLENDING: For partially-masked pixels (mask value between 1-254), blend between original and adjusted pixel using mask value as alpha:
      ```python
      alpha = mask_pixel / 255.0
      result_pixel = (1 - alpha) * original + alpha * adjusted
      ```
      This is what makes feathered mask edges seamless.
    - Use numpy for performance if image is large (>1024x1024), fall back to pixel-by-pixel for smaller images
    - Return PNG bytes

    **3. blend_seams(image_bytes, seam_pixels, blend_radius=6) -> bytes** (TEX-05)
    - Accept image bytes and list of seam pixel coordinates `[(x, y), ...]`
    - For each seam pixel, sample colors on both sides of the seam within `blend_radius`
    - Apply distance-weighted Gaussian blend: pixels closer to seam get more blending, pixels at blend_radius distance get zero blending
    - Implementation: Create a seam mask (L-mode) with 255 at seam pixels, blur by `blend_radius`, then use `Image.composite(blurred_image, original_image, seam_mask)` where `blurred_image` is the Gaussian-blurred version of the original
    - This produces a smooth color transition across seam boundaries without hard edges
    - Return PNG bytes

    **4. make_tileable(image_bytes, overlap_pct=0.15) -> bytes** (TEX-09)
    - Load image from bytes
    - Calculate overlap region: `overlap = int(width * overlap_pct)`
    - Cross-fade method:
      - Extract left/right/top/bottom strips of `overlap` width
      - Create linear gradient masks (0 to 255 across the overlap width)
      - Mirror the opposite edge strip and blend with gradient mask
      - For corners: blend diagonally using 2D gradient
    - Result: left edge pixels match right edge pixels, top matches bottom
    - Return PNG bytes

    **5. render_wear_map(curvature_data, texture_size, uv_data=None) -> bytes** (TEX-06 helper)
    - Accept curvature per-vertex dict `{vertex_index: float}` and texture_size
    - If uv_data provided (list of face UV polygons with vertex indices), render curvature values to UV space
    - Normalize curvature to 0-1 range: convex (positive) = bright, concave (negative) = dark
    - Create L-mode image and fill polygons with interpolated curvature values
    - Return PNG bytes

    **6. inpaint_texture(image_bytes, mask_bytes, prompt, fal_key=None) -> dict** (TEX-03 stub)
    - If `fal_key` is None or empty, return `{"status": "stub", "message": "fal.ai API key not configured"}`
    - If key available: use fal_client to call FLUX inpainting endpoint (full implementation in Plan 03-04)
    - For now, this is a placeholder that validates inputs and returns stub status

    - Feathered mask test: create mask with single polygon, verify edge pixels are in range 1-254 (not hard 0 or 255)

    - Tileable test: apply make_tileable, tile the result 2x2, verify pixel diff at seam lines is below threshold
