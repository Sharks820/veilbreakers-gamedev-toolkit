    ```python
    @mcp.tool()
    async def blender_texture(
        action: Literal[
            "create_pbr", "mask_region", "inpaint", "hsv_adjust",
            "blend_seams", "generate_wear", "bake", "upscale",
            "make_tileable", "validate"
        ],
        object_name: str | None = None,
        # PBR params
        material_name: str | None = None,
        texture_dir: str | None = None,
        texture_size: int = 1024,
        # Mask params
        material_index: int | None = None,
        feather_radius: int = 5,
        # HSV params
        image_path: str | None = None,
        mask_path: str | None = None,
        hue_shift: float = 0.0,
        saturation_scale: float = 1.0,
        value_scale: float = 1.0,
        # Bake params
        bake_type: str = "NORMAL",
        source_object: str | None = None,
        image_name: str | None = None,
        margin: int = 16,
        cage_extrusion: float = 0.1,
        samples: int = 32,
        # Upscale params
        scale: int = 4,
        model: str = "realesrgan-x4plus",
        # Tileable params
        overlap_pct: float = 0.15,
        # Inpaint params
        prompt: str | None = None,
        # Seam blend params
        blend_radius: int = 6,
        capture_viewport: bool = True,
    ):
        """Texture creation, editing, baking, upscaling, and validation.

        Actions:
        - create_pbr: Create full PBR material with image textures (albedo, normal, roughness, metallic, AO)
        - mask_region: Generate UV mask for a material/region with feathered edges for surgical editing
        - inpaint: AI texture inpainting on masked region (requires FAL_KEY)
        - hsv_adjust: Adjust hue/saturation/value on masked texture region with seamless blending
        - blend_seams: Blend color discontinuities along UV island seam boundaries
        - generate_wear: Generate procedural wear/damage map from mesh curvature
        - bake: Bake texture maps (normal, AO, combined) from high-poly to low-poly
        - upscale: AI texture upscaling via Real-ESRGAN (2x/4x)
        - make_tileable: Convert texture to seamless tileable version
        - validate: Check texture resolution, format, UV coverage, compression suitability
        """
    ```

    Routing logic:
    - `create_pbr`: dispatch to Blender `texture_create_pbr` via send_command, return with screenshot
    - `mask_region`: dispatch to Blender `texture_get_uv_region` to get UV polygons, then call `texture_ops.generate_uv_mask()` MCP-side, return mask as Image
    - `inpaint`: call `texture_ops.inpaint_texture()` with image/mask bytes and prompt
    - `hsv_adjust`: load image and mask from paths, call `texture_ops.apply_hsv_adjustment()`, save result, return as Image
    - `blend_seams`: dispatch to Blender `texture_get_seam_pixels` for seam coords, load texture, call `texture_ops.blend_seams()`, save, return as Image
    - `generate_wear`: dispatch to Blender `texture_generate_wear` for curvature data, call `texture_ops.render_wear_map()` MCP-side, return as Image
    - `bake`: dispatch to Blender `texture_bake`, return with screenshot
    - `upscale`: call `esrgan_runner.upscale_texture()`, return result path
    - `make_tileable`: load image, call `texture_ops.make_tileable()`, save, return as Image
    - `validate`: dispatch to Blender `texture_validate`, return JSON
