# Phase 33 Plan 01: Interior System -- Spatial-Aware Furnishing

## Objective

Transform `generate_interior_layout()` from random placement into a constraint-satisfied, spatially-aware interior generation system with relationship graphs, clutter scatter, and practical lighting. All pure-logic (no bpy) in `_building_grammar.py` for testability.

## Context

@33-RESEARCH.md @33-CONTEXT.md @.planning/REQUIREMENTS.md (MESH-03)

## Tasks

### Task 1: Room Spatial Graphs + Constraint Solver
type="auto"

**What:** Add `ROOM_SPATIAL_GRAPHS` defining per-room-type spatial relationships (focal points, item clusters, wall preferences) and implement a constraint satisfaction post-processor that refines random placements into spatially coherent layouts.

**Implementation:**
1. Define `ROOM_SPATIAL_GRAPHS` dict mapping room types to relationship definitions:
   - `focal_points`: items that anchor room zones (fireplace, altar, forge, bar_counter)
   - `clusters`: groups of items that belong together (table + chairs, bed + nightstand)
   - `wall_preferences`: which walls focal items prefer (fireplace=back, bar=back, forge=exterior)
2. Replace `generate_interior_layout` with enhanced version:
   - Phase 1: Place focal points on preferred walls
   - Phase 2: Place clustered items relative to their anchors (chairs around tables, stools at bar)
   - Phase 3: Place remaining wall/corner items with wall clearance 0.3m
   - Phase 4: Enforce door clearance corridor (1.0m wide path from each wall's door zone to room center)
   - All items face appropriate direction (chairs toward table center, beds toward room center)
3. Collision detection remains AABB but with 0.3m wall margin (up from 0.15m)

**Done criteria:**
- `ROOM_SPATIAL_GRAPHS` covers at least tavern, bedroom, kitchen, blacksmith, library, chapel, throne_room
- Chairs face their associated table (rotation points toward table center)
- Nightstands placed within 0.5m of bed head
- 1.0m clear corridor from door zone to room center
- Wall items have 0.3m clearance from room edges
- All existing tests still pass

### Task 2: Activity Zone Templates
type="auto"

**What:** Define activity zones per room type that partition the room into functional areas, ensuring furniture falls within appropriate zones.

**Implementation:**
1. Add `ROOM_ACTIVITY_ZONES` mapping room types to zone definitions:
   - Each zone has: name, fraction of room (e.g., bar_zone=0.3 of width along back wall), allowed item types
   - Tavern: bar_zone (back wall, 30%), seating_zone (center, 50%), hearth_zone (side wall, 20%)
   - Kitchen: fire_zone (back wall), prep_zone (center), storage_zone (corners)
   - Bedroom: sleep_zone (back wall), work_zone (window wall), storage_zone (side wall)
   - Blacksmith: forge_zone (back wall), anvil_zone (center-front), storage_zone (sides)
2. Integrate zone assignment into the constraint solver from Task 1
3. Items are placed preferentially within their assigned zone

**Done criteria:**
- Activity zones defined for tavern, bedroom, kitchen, blacksmith, library, chapel
- Items land in correct zones (bar_counter in bar_zone, tables in seating_zone)
- Zone coverage >= 80% of room area (no dead zones)
- Tests verify zone assignment for key room types

### Task 3: Decorative Clutter Scatter
type="auto"

**What:** Implement Poisson disk scatter system that places 5-15 decorative props per room on furniture surfaces and floor areas.

**Implementation:**
1. Define `CLUTTER_POOLS` per room type:
   - Tavern: mug, plate, bottle, candle_stub, coin_pile, food_scrap, spilled_drink
   - Bedroom: book, candle, mirror, clothing_pile, comb, jewelry_box
   - Kitchen: pot, ladle, cutting_board, vegetable, flour_bag, herb_bunch
   - Blacksmith: hammer, tongs, horseshoe, metal_ingot, grinding_stone, coal_pile
   - Library: open_book, quill_ink, scroll, magnifying_glass, globe_small
   - Chapel: prayer_bead, incense_holder, offering_bowl, holy_water_vial
2. Implement `_poisson_disk_scatter_2d(width, depth, min_distance, rng)` returning sample points
3. Implement `generate_clutter_layout(room_type, width, depth, furniture_items, seed, density=0.5)`:
   - Identify scatter surfaces from furniture bounding boxes (table tops, shelf surfaces, floor zones)
   - Sample points using Poisson disk with min_distance=0.15-0.3m
   - Select items from room-type clutter pool
   - Scale variation +/-15%
   - Return list of clutter items with position, rotation, scale, surface_parent
4. Density parameter (0.0-1.0) controls item count: floor(5 + density * 10) items

**Done criteria:**
- Clutter pools defined for at least 6 room types
- Poisson disk sampling produces non-overlapping points
- 5-15 clutter items per room at default density
- Clutter items placed on furniture surfaces (table tops) and floor
- Same seed produces same clutter layout
- No clutter items outside room bounds

### Task 4: Lighting Placement Engine
type="auto"

**What:** Add intelligent lighting placement producing light source data for each room with appropriate color temperature and coverage.

**Implementation:**
1. Define `LIGHTING_SCHEMAS` per room type:
   - Each schema lists mandatory light sources and conditional sources
   - All rooms: min 2 light sources
   - Light types: torch_sconce (2800K, 4m), candle (3000K, 2m), fireplace_light (2700K, 5m), chandelier_light (3200K, 8m), brazier_light (3000K, 3m)
2. Implement `generate_lighting_layout(room_type, width, depth, height, furniture_items, door_positions, seed)`:
   - Place torches at doorway positions (1.6m height, both sides)
   - Place candles on table surfaces
   - Add fireplace emissive point light if fireplace exists
   - Add chandelier light at ceiling center if room > 8m in any dimension
   - Ensure min 2 light sources, add supplemental wall torches if needed
   - Return list of light sources with position, type, color_temperature, radius, intensity
3. Color temperature range: 2700K-3500K (warm interior lighting)
4. Update `handle_generate_interior` in `worldbuilding.py` to consume light source data and create Blender point lights

**Done criteria:**
- Lighting schemas defined for all 22 room types
- Min 2 light sources per room guaranteed
- All light sources in 2700-3500K range
- Torches placed at doorway positions
- Candles on tables, fireplace emissive present
- Light placement deterministic with seed
- Tests verify light count, temperature range, and doorway placement
