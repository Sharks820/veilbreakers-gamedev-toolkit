  <action>
1. In visual_validation.py, add function `aaa_verify_map(screenshot_paths: list[str], min_score: int = 60) -> dict` that:
   - Takes a list of screenshot file paths (one per camera angle, expect 10)
   - Calls `analyze_render_image()` on each
   - Adds TWO new quality checks per image:
     a. **Floating geometry detection**: Check if >5% of pixels in the bottom 20% of the image are sky-colored (mean brightness > 200 in a region that should be ground). Flag as "floating_geometry_suspected".
     b. **Default material detection**: Check if the image has very low color variance (std_dev < 8 across all channels) indicating a single gray/white default material. Flag as "default_material_detected".
   - Returns `{passed: bool, total_score: float (avg of all angles), per_angle: [{angle_id, score, issues}], failed_angles: [ids where score < min_score]}`
   - An angle passes if: score >= min_score AND no "default_material_detected" AND no "floating_geometry_suspected"

2. In screenshot_diff.py, add function `capture_regression_baseline(screenshot_paths: list[str], baseline_dir: str) -> dict` that:
   - Copies each screenshot to baseline_dir with naming convention `baseline_{angle_id}.png`
   - Returns `{baseline_count: int, baseline_dir: str, paths: [str]}`

3. In blender_server.py, add action `"aaa_verify"` to the asset_pipeline tool (add to the actions list near line 2046):
   - Accepts params: `angles` (int, default 10), `min_score` (int, default 60), `capture_baseline` (bool, default False)
   - Implementation: sends Blender commands to render 10 angles (front, back, left, right, top, 45-degree NE/NW/SE/SW, ground-level) by rotating camera around scene center
   - Collects screenshot paths, calls `aaa_verify_map()`
   - If `capture_baseline=True`, also calls `capture_regression_baseline()`
   - Returns full verification results plus any screenshots

4. Also add action `"screenshot_regression"` to asset_pipeline:
   - Accepts: `baseline_dir`, `current_screenshots` list
   - Calls `compare_screenshots()` for each pair
   - Returns pass/fail with diff percentages per angle
  </action>
