   - Instead of flat disc, create a subdivided mesh that follows river/lake spline path
   - Parameters: `path_points` (list of (x,y,z)), `width` (default 8m for river, variable for lake)
   - Cross-section: 8-16 subdivisions perpendicular to flow direction
   - **Shore blending**: vertices within 0.5m of shore get alpha gradient (1.0 at center to 0.0 at edge). Vertices where depth < 0.2m get foam flag.
   - **Flow vertex colors** per research spec:
     - R = flow speed (0.0 still to 1.0 fast, based on channel width -- narrower = faster)
     - G = flow direction X component (normalized, remapped 0-1)
     - B = flow direction Z component (normalized, remapped 0-1)
     - A = foam intensity (1.0 where depth < 0.2m or flow speed > 0.8, else 0.0)
   - Apply transparent water material: sRGB(40, 60, 50) base, roughness 0.05, alpha 0.6, IOR 1.33
   - Mesh budget: keep under 20K tris (per performance spec)

2. **Add ridged multifractal noise** (_terrain_noise.py):
   - Add function `ridged_multifractal(x, y, octaves=6, lacunarity=2.0, gain=0.5, offset=1.0, seed=42) -> float`:
     - Implementation: for each octave, compute noise, invert (offset - abs(noise)), square the result, accumulate weighted by frequency
     - This produces sharp ridgelines and mountain peaks (vs smooth Perlin hills)
     - Much better for dark fantasy craggy terrain
   - Wire into terrain generation: add `noise_type` param to terrain handler, options: "perlin" (default, existing), "ridged_multifractal" (new), "hybrid" (blend of both)

3. **Upgrade terrain auto-splatting** (environment.py or _terrain_noise.py):
   - Add function `auto_splat_terrain(terrain_obj, biome_map) -> dict`:
     - For each terrain vertex, compute:
       a. Slope angle (from face normal vs world up)
       b. Height (normalized 0-1 across terrain range)
       c. Curvature (convex = ridge, concave = valley)
       d. Moisture (proximity to water sources + height-based rainfall)
     - Assign splat weights per biome rules from research:
       - slope > 55 deg -> cliff/rock (100%)
       - slope 30-55 deg -> rock/gravel blend
       - height > 0.7 -> mountain/snow
       - moisture > 0.6 AND slope < 10 -> swamp/mud
       - else -> grass/dirt blend based on biome
     - Apply curvature-based roughness: convex edges -0.15 roughness, concave +0.2
     - Output: vertex colors encoding splat weights (can be baked to splat map texture)
