<objective>
Build the Blender addon with TCP socket server and the security validation module. The addon receives commands from the MCP server over TCP, queues them for main-thread execution via bpy.app.timers, dispatches to typed handler functions, and returns structured JSON responses. The security module provides AST-based code validation that blocks dangerous imports and functions before any Python code reaches exec().

Purpose: The addon is the other half of the communication bridge -- without it, the MCP server has nothing to talk to. The queue+timer pattern is the single most critical safety mechanism (prevents segfaults from threading violations). The security module prevents prompt injection attacks through the code execution tool.

Output: A Blender addon that registers cleanly, starts a socket server, dispatches commands to handlers on the main thread, and a security.py module that validates Python code against an import whitelist.
</objective>

    2. Create `Tools/mcp-toolkit/blender_addon/socket_server.py`:
       Class `BlenderMCPServer`:
       - `__init__(self, port: int = 9876)`:
         `self.port = port`, `self.command_queue = queue.Queue()`, `self.server_thread = None`, `self.running = False`, `self._server_socket = None`
       - `def start(self)`:
         Sets `self.running = True`. Creates daemon thread `_server_loop`. Registers `bpy.app.timers.register(self._process_commands, first_interval=0.05, persistent=True)`
       - `def stop(self)`:
         Sets `self.running = False`. Closes `_server_socket` if open. Unregisters timer via `bpy.app.timers.is_registered(self._process_commands)` check then `bpy.app.timers.unregister(self._process_commands)`. Joins thread with timeout=2.0.
       - `def _server_loop(self)` (background thread -- NO bpy calls):
         Creates `socket.AF_INET, socket.SOCK_STREAM`, `setsockopt(SO_REUSEADDR, 1)`, `bind(("localhost", self.port))`, `listen(1)`, `settimeout(1.0)`. Loop: accept client, handle in `_handle_client`. Catch `socket.timeout` to continue. On stop, close socket.
       - `def _handle_client(self, client_sock)` (background thread -- NO bpy calls):
         Read 4-byte length prefix via `struct.unpack(">I", ...)`, then read that many bytes for JSON payload. Parse JSON. Create `threading.Event()` and `result_container = {}`. Put `(command_dict, result_event, result_container)` on `self.command_queue`. `result_event.wait(timeout=300)`. If timed out, send `{"status": "error", "message": "Command execution timed out"}`. Otherwise send `result_container["response"]` back with same length-prefix protocol. Close client socket.
       - `def _receive_exactly(self, sock, n)` helper: loop recv until n bytes accumulated.
       - `def _process_commands(self)` (MAIN THREAD via timer -- safe for bpy):
         Drain queue: `while not self.command_queue.empty()`. For each: get `(cmd, event, container)`. Look up `cmd["type"]` in `COMMAND_HANDLERS` dict. If found, call `handler(cmd.get("params", {}))`. Wrap in try/except to catch any exception and return `{"status": "error", "message": str(e)}`. Set `container["response"]` and `event.set()`. Return `0.05` to re-register timer.

       ALLOWED_IMPORTS = frozenset({
           "bpy", "mathutils", "bmesh", "math", "random", "json",

           "os", "sys", "subprocess", "socket", "http", "urllib",
           "shutil", "ctypes", "importlib", "pathlib", "io",
           "pickle", "shelve", "tempfile", "glob", "fnmatch",

       })

       Class `SecurityValidator(ast.NodeVisitor)`:
       - `__init__(self)`: `self.violations: list[str] = []`
       - `visit_Import(self, node)`: For each alias, extract root module. If in `BLOCKED_IMPORTS`, append violation. If not in allowed root set, append "Unknown import" violation. Call `self.generic_visit(node)`.
       - `visit_ImportFrom(self, node)`: If `node.module` root is in `BLOCKED_IMPORTS`, append violation. Call `self.generic_visit(node)`.
       - `visit_Call(self, node)`: Block `exec`, `eval`, `compile`, `getattr`, `setattr`, `delattr` by checking `isinstance(node.func, ast.Name) and node.func.id in (...)`. Append violation. Call `self.generic_visit(node)`.
       - `visit_Attribute(self, node)`: Block access to dunder attributes like `__class__`, `__bases__`, `__subclasses__`, `__import__`, `__builtins__`. If `node.attr.startswith("__") and node.attr.endswith("__")` and attr is in a blocked set, append violation.

       Function `validate_code(code: str) -> tuple[bool, list[str]]`:
       - Try `ast.parse(code)`. On `SyntaxError`, return `(False, [f"Syntax error: {e}"])`.
       - Create `SecurityValidator`, call `validator.visit(tree)`.
       - Return `(len(validator.violations) == 0, validator.violations)`.
