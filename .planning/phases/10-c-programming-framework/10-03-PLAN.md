Generate static `ServiceLocator` class:
- `private static readonly Dictionary<Type, object> _services = new();`
- `Register<T>(T service)` -- store by typeof(T)
- `Get<T>()` -- retrieve with `InvalidOperationException` if not found
- `TryGet<T>(out T service)` -- safe retrieval returning bool

- `Clear()` -- reset all
