Implement handler for each action:
- `"generate_class"`: Call `generate_class(class_name, class_type, namespace, ...)`, write to `{output_dir}/{class_name}.cs`. Validate class_name is not empty.
- `"modify_script"`: Read file at `script_path` from Unity project using `Path(settings.unity_project_path) / script_path`. Create `.cs.bak` backup. Call `modify_script(source, ...)`. Write modified source back. Return changes list in response.
- `"editor_window"`: Call `generate_editor_window(window_name, menu_path, namespace, ...)`, write to `Assets/Editor/Generated/Code/{window_name}.cs`
- `"property_drawer"`: Call `generate_property_drawer(target_type, namespace, drawer_body)`, write to `Assets/Editor/Generated/Code/{target_type}Drawer.cs`
- `"inspector_drawer"`: Call `generate_inspector_drawer(target_type, namespace, fields_to_draw)`, write to `Assets/Editor/Generated/Code/{target_type}Editor.cs`
- `"scene_overlay"`: Call `generate_scene_overlay(overlay_name, display_name, namespace, panel_body)`, write to `Assets/Editor/Generated/Code/{overlay_name}Overlay.cs`
- `"generate_test"`: Call `generate_test_class(class_name, test_mode, namespace, target_class, test_methods, setup_body, teardown_body)`, write to `Assets/Tests/{test_mode}/{class_name}.cs`
- `"service_locator"`: Call `generate_service_locator(namespace, include_scene_persistent)`, write to `{output_dir}/ServiceLocator.cs`
- `"object_pool"`: Call `generate_object_pool(namespace, include_gameobject_pool)`, write to `{output_dir}/ObjectPool.cs`
- `"singleton"`: Call `generate_singleton(class_name, singleton_type, namespace, persistent)`, write to `{output_dir}/{class_name}.cs`
- `"state_machine"`: Call `generate_state_machine(namespace)`, write to `{output_dir}/StateMachine.cs`
- `"event_channel"`: Call `generate_so_event_channel(event_name, has_parameter, parameter_type, namespace)`, write to `{output_dir}/GameEvent.cs` (or `{event_name}Event.cs` if specific)

All actions return JSON with `status`, `action`, `script_path`, `next_steps` (always includes "Call unity_editor action='recompile'").

For `modify_script`, include `backup_path` and `changes` in the response.
