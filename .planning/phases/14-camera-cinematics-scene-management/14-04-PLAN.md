Add these 9 generator functions, all producing runtime MonoBehaviours in VeilBreakers.WorldSystems namespace plus editor setup scripts:

1. **generate_fast_travel_script** (RPG-02): Params: waypoint_prefab_path (str), teleport_fade_duration (float=0.5), save_key (str="discoveredWaypoints"), namespace. Returns tuple (editor_cs, runtime_cs). Runtime: VB_WaypointManager MonoBehaviour. OnTriggerEnter discovers waypoint (adds to List<string>, invokes OnWaypointDiscovered event). TeleportTo method with loading transition. Save/load discovered list via JsonUtility. Editor: setup script. [MenuItem("VeilBreakers/World/Setup Fast Travel")].

2. **generate_puzzle_mechanics_script** (RPG-04): Params: puzzle_types (list of str, default ["lever_sequence", "pressure_plate", "key_lock", "light_beam"]), namespace. Returns tuple (editor_cs, runtime_cs). Runtime: abstract PuzzleMechanic base + 4 subclasses. PuzzleMechanic has IsSolved property, OnSolved UnityEvent, ResetPuzzle(). LeverSequencePuzzle: ordered lever activation. PressurePlatePuzzle: weighted plate triggers. KeyLockPuzzle: inventory key check. LightBeamPuzzle: directional light alignment. Editor: spawns puzzle prefab. [MenuItem("VeilBreakers/World/Create Puzzle")].

3. **generate_trap_system_script** (RPG-06): Params: trap_types (list of str, default ["pressure_plate", "dart_wall", "spike_pit", "poison_gas", "swinging_blade"]), base_damage (float=25.0), cooldown (float=3.0), namespace. Returns tuple (editor_cs, runtime_cs). Runtime: abstract TrapBase with Activate/Reset methods, _damage, _cooldown fields. 5 subclasses with unique activation behaviors. PressurePlateTrap: OnTriggerEnter. DartWallTrap: projectile spawn. SpikePitTrap: falling detection. PoisonGasTrap: area damage over time. SwingingBladeTrap: pendulum with collision. [MenuItem("VeilBreakers/World/Create Trap")].

4. **generate_spatial_loot_script** (RPG-07): Params: chest_prefab_path (str), loot_table_so_path (str), room_loot_density (float=0.3), namespace. Returns tuple (editor_cs, runtime_cs). Runtime: VB_SpatialLootManager. Treasure chest positions as Transform children. Room-based drop tables via SO reference. Treasure room detection (tag/name). Random item selection weighted by room type. Editor: place loot markers. [MenuItem("VeilBreakers/World/Setup Spatial Loot")].

5. **generate_weather_system_script** (RPG-09): Params: weather_states (list, default ["Clear","Rain","Snow","Fog","Storm"]), transition_duration (float=3.0), default_state (str="Clear"), namespace. Returns tuple (editor_cs, runtime_cs). Runtime: VB_WeatherManager. WeatherState enum. ParticleSystem references per state. TransitionTo(WeatherState) via coroutine: lerp emission.rateOverTime from current to 0, then from 0 to target. Adjust fog density/color. Optional gameplay effects callback. Editor: creates weather manager GO with particle system children. [MenuItem("VeilBreakers/World/Create Weather System")].

6. **generate_day_night_cycle_script** (RPG-10): Params: day_duration_minutes (float=10.0), start_hour (float=8.0), time_presets (list of 8 preset names), namespace. Returns tuple (editor_cs, runtime_cs). Runtime: VB_DayNightCycleManager. Continuous _timeOfDay float (0-24) in Update. Lerp between time-of-day presets for directional light rotation, color, intensity, ambient, fog. UnityEvent<float> OnTimeChanged for NPC schedules. UnityEvent<bool> OnNightfall (hour>=20), OnDaybreak (hour>=6). CurrentHour property. SetTime/PauseTime methods. Editor: creates cycle manager with directional light. [MenuItem("VeilBreakers/World/Create Day Night Cycle")].

7. **generate_npc_placement_script** (RPG-11): Params: npc_roles (list, default ["shopkeeper","quest_giver","bartender","guard"]), namespace. Returns tuple (so_cs, runtime_cs, editor_cs). SO: VB_NPCPlacementData ScriptableObject with NPCSlot[] array (role enum, position, rotation, prefab_path). Runtime: VB_NPCPlacementManager that reads SO data and instantiates NPCs at positions on scene load. Editor: creates SO asset + placement manager. [MenuItem("VeilBreakers/World/Setup NPC Placement")].

8. **generate_dungeon_lighting_script** (RPG-12): Params: torch_spacing (float=5.0), torch_light_range (float=8.0), torch_color ([1.0,0.7,0.3,1.0]), fog_density (float=0.03), fog_color ([0.1,0.08,0.06]), namespace. Returns tuple (editor_cs, runtime_cs). Runtime: VB_DungeonLightingSetup. Places torch sconces every 4-6m along corridor paths (configurable spacing). Point light per torch with range, warm orange color, soft shadows. Atmospheric fog zones with volumetric settings. Dark corridor baseline with light pools at fire sources. Editor: auto-places torches along selected path. [MenuItem("VeilBreakers/World/Setup Dungeon Lighting")].

9. **generate_terrain_building_blend_script** (RPG-13): Params: blend_radius (float=2.0), decal_material_path (str), depression_depth (float=0.1), vertex_color_falloff (float=1.5), namespace. Returns tuple (editor_cs, runtime_cs). Runtime: VB_TerrainBuildingBlend. Paints vertex color at building foundation base (dark/dirt color blending into terrain). Places DecalProjector for ground-to-building transition. Depresses terrain height beneath structure via TerrainData.SetHeights. Editor: applies blend to selected building. [MenuItem("VeilBreakers/World/Blend Terrain Building")].

All generators use line-based string concatenation, _sanitize helpers. Most return tuples (editor_cs, runtime_cs). NPC placement returns triple (so_cs, runtime_cs, editor_cs). Update __all__ list.
