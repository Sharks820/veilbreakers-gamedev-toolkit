// Scene creation/loading

StartCoroutine(LoadSceneRoutine(sceneName, mode));

Implement these 9 generator functions:

1. **generate_scene_creation_script** (SCNE-01): Params: scene_name, scene_setup (str="DefaultGameObjects" or "EmptyScene"), loading_mode (str="single" or "additive"), build_index (int=-1), namespace. Editor script that creates scene via EditorSceneManager.NewScene, optionally adds to Build Settings. Runtime loading helper uses SceneManager.LoadSceneAsync with explicit LoadSceneMode. [MenuItem("VeilBreakers/World/Create Scene")].

2. **generate_scene_transition_script** (SCNE-02): Params: fade_duration (float=0.5), show_loading_screen (bool=True), namespace. Runtime MonoBehaviour in VeilBreakers.WorldSystems namespace. Singleton with DontDestroyOnLoad. Coroutine-based: FadeOut -> LoadSceneAsync (allowSceneActivation=false, progress tracking) -> FadeIn. Supports VeilBreakers flow: Bootstrap->MainMenu->CharacterSelect->Overworld<->Battle. Returns tuple (editor_cs, runtime_cs) -- editor script sets up the manager prefab, runtime is the MonoBehaviour.

3. **generate_probe_setup_script** (SCNE-03): Params: reflection_probe_count (int=4), reflection_resolution (int=256), probe_box_size ([10,5,10]), light_probe_grid_spacing (float=2.0), light_probe_grid_size ([5,3,5]), namespace. Creates ReflectionProbe components with mode/resolution/boxSize. Creates LightProbeGroup with programmatic grid positions. [MenuItem("VeilBreakers/World/Setup Probes")].

4. **generate_occlusion_setup_script** (SCNE-04): Params: smallest_occluder (float=5.0), smallest_hole (float=0.25), backface_threshold (float=100.0), namespace. Marks selected objects with StaticEditorFlags.OccluderStatic or OccludeeStatic based on renderer bounds. Calls StaticOcclusionCulling.Compute(). [MenuItem("VeilBreakers/World/Setup Occlusion")].

5. **generate_environment_setup_script** (SCNE-05): Params: skybox_shader (str="Skybox/Procedural"), ambient_mode (str="Skybox"), default_reflection_mode (str="Skybox"), enable_gi (bool=True), namespace. Configures RenderSettings (skybox material, ambientMode, defaultReflectionMode). Optionally triggers Lightmapping.BakeAsync(). [MenuItem("VeilBreakers/World/Setup Environment")].

6. **generate_terrain_detail_script** (SCNE-06): Params: detail_prototypes (list of dicts with type: "grass_texture" or "detail_mesh", texture_path or prefab_path, min_height, max_height, min_width, max_width, color), paint_density (int=8), namespace. Creates DetailPrototype array on TerrainData, paints density maps via SetDetailLayer. [MenuItem("VeilBreakers/World/Paint Terrain Detail")].

7. **generate_tilemap_setup_script** (TWO-01): Params: grid_cell_size ([1,1,0]), tile_entries (list of dicts with x, y, tile_asset_path), rule_tile_name (str, optional), rule_tile_rules (list of dicts for tiling rules, optional), namespace. Creates Tilemap on Grid, places tiles via SetTile. Optionally creates RuleTile ScriptableObject with tiling rules. [MenuItem("VeilBreakers/World/Setup Tilemap")].

8. **generate_2d_physics_script** (TWO-02): Params: gravity ([0,-9.81]), collider_type (str="box" or "circle" or "composite"), body_type (str="Dynamic" or "Kinematic" or "Static"), joint_type (str, optional: "hinge" or "spring" or "distance"), joint_params (dict, optional), namespace. Configures Physics2D.gravity, adds Rigidbody2D + Collider2D + optional joint. [MenuItem("VeilBreakers/World/Setup 2D Physics")].

9. **generate_time_of_day_preset_script** (WORLD-08): Params: preset_name (str from 8 presets), custom_overrides (dict, optional for sun_color, fog_color, etc.), apply_fog (bool=True), namespace. Uses _WORLD_TIME_PRESETS dict. Generates C# that applies full scene lighting from preset (sun rotation, color, intensity, ambient, fog, skybox tint). Returns editor script. [MenuItem("VeilBreakers/World/Apply Time of Day")].

All generators: line-based string concatenation, _sanitize helpers, module docstring, __all__ list. Note: generate_scene_transition_script returns a tuple (editor_cs, runtime_cs) -- document this in docstring.
