1. **generate_cinemachine_setup_script** (CAM-01): Params: camera_type (orbital/follow/dolly), follow_target, look_at_target, priority (int=10), radius (float=5.0), target_offset ([0,1.5,0]), damping ([1,0.5,0]), namespace. Static class with [MenuItem("VeilBreakers/Camera/Setup Virtual Camera")].

2. **generate_state_driven_camera_script** (CAM-01): Params: camera_name, states (list of dicts with state_name, camera_name), namespace. Creates CinemachineStateDrivenCamera with AnimatedTarget assignment. Instructions for child camera creation. [MenuItem("VeilBreakers/Camera/Setup State-Driven Camera")].

3. **generate_camera_shake_script** (CAM-04): Params: impulse_force (float=0.5), impulse_duration (float=0.2), add_listener (bool=True), namespace. Creates CinemachineImpulseSource on target + CinemachineImpulseListener on main camera. [MenuItem("VeilBreakers/Camera/Setup Camera Shake")].

4. **generate_camera_blend_script** (CAM-04): Params: default_blend_time (float=2.0), blend_style (str="EaseInOut"), custom_blends (list of dicts with from_cam, to_cam, time, style), namespace. Configures CinemachineBrain default blend + custom blend list. [MenuItem("VeilBreakers/Camera/Configure Blend")].

5. **generate_timeline_setup_script** (CAM-02): Params: timeline_name, tracks (list of dicts with type: animation/audio/activation/cinemachine and name), output_path (str="Assets/Timelines"), namespace. Creates TimelineAsset, saves to AssetDatabase FIRST, then calls CreateTrack<T> for each track. Import `using UnityEngine.Timeline; using UnityEngine.Playables; using Unity.Cinemachine;`. [MenuItem("VeilBreakers/Camera/Create Timeline")].

6. **generate_cutscene_setup_script** (CAM-03): Params: cutscene_name, timeline_path, wrap_mode (str="None"), play_on_awake (bool=False), namespace. Creates PlayableDirector, assigns timeline asset, configures wrap mode and play-on-awake. [MenuItem("VeilBreakers/Camera/Setup Cutscene")].

7. **generate_animation_clip_editor_script** (ANIMA-01): Params: clip_name, curves (list of dicts with path, component_type, property, keyframes: list of [time, value]), output_path (str="Assets/Animations"), namespace. Uses EditorCurveBinding.FloatCurve and AnimationUtility.SetEditorCurve (NOT AnimationClip.SetCurve). [MenuItem("VeilBreakers/Camera/Edit Animation Clip")].

8. **generate_animator_modifier_script** (ANIMA-02): Params: controller_path, states_to_add (list of str), transitions (list of dicts with from_state, to_state, has_exit_time, duration), parameters (list of dicts with name, type: Float/Int/Bool/Trigger), sub_state_machines (list of str), namespace. Uses AnimatorController from UnityEditor.Animations; layers[0].stateMachine for state/transition management. [MenuItem("VeilBreakers/Camera/Modify Animator")].

9. **generate_avatar_mask_script** (ANIMA-03): Params: mask_name, body_parts (dict mapping part name to bool, e.g. {"Head": True, "LeftLeg": False}), transform_paths (list of str, optional for non-humanoid), output_path (str="Assets/Animations"), namespace. Uses AvatarMask.SetHumanoidBodyPartActive(AvatarMaskBodyPart.X, bool). [MenuItem("VeilBreakers/Camera/Create Avatar Mask")].

10. **generate_video_player_script** (MEDIA-01): Params: video_source (str="clip" or "url"), video_path, render_texture_width (int=1920), render_texture_height (int=1080), loop (bool=True), play_on_awake (bool=True), namespace. Creates VideoPlayer + RenderTexture, configures renderMode. [MenuItem("VeilBreakers/Camera/Setup Video Player")].

All generators: line-based string concatenation (`lines = []`, `lines.append(...)`, `return "\\n".join(lines)`). Use _sanitize_cs_string for all string params, _sanitize_cs_identifier for names. Each returns a single C# string. Add module-level docstring and __all__ list.
