    **shader_templates.py** -- 7 generator functions. Each returns HLSL ShaderLab source:
    - All follow: `Shader "VeilBreakers/{Name}" { Properties { ... } SubShader { ... Pass { HLSLPROGRAM ... ENDHLSL } } }`
    - All include URP core: `#include "Packages/com.unity.render-pipelines.universal/ShaderLibrary/Core.hlsl"`
    - Corruption shader: lerps base color toward corruption color based on _CorruptionAmount; adds noise-based vein pattern
    - Dissolve: noise-based clip with edge glow (per RESEARCH code example)
    - Force field: fresnel + intersection highlight using scene depth
    - Water: vertex displacement + transparency + refraction approximation
    - Foliage: wind sway on vertices using _Time and sin/cos, subsurface scattering approximation
    - Outline: two-pass (enlarged backface + normal render) or screen-space Sobel edge detection
    - Damage overlay: fullscreen pass with blood texture overlay, alpha controlled by _Intensity
