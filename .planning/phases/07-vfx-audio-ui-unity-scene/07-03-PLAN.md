      - generate_sfx calls `client.text_to_sound_effects.convert(text=description, duration_seconds=duration_seconds, prompt_influence=0.5)`
      - Collects bytes from iterator, writes to output_path
      - generate_music_loop calls same SFX endpoint with "loopable " prefix in text, writes .mp3
      - generate_voice_line calls `client.text_to_speech.convert(text=text, voice_id=voice_id, model_id="eleven_multilingual_v2")`
      - generate_ambient_layers calls generate_sfx for each layer description (e.g., "wind" + "birds" + "water" for forest biome)
    - Retry logic: catch HTTP 429, retry 3 times with 2s/4s/8s exponential backoff
    - All methods return dict with path, duration, stub flag, and description
