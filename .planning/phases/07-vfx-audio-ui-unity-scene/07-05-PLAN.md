Terrain setup:

```csharp
var terrainData = new TerrainData();

terrainData.size = new Vector3(1000, 600, 1000);

var heights = new float[513, 513];
// ... read from file
terrainData.SetHeights(0, 0, heights);
terrainData.SetAlphamaps(0, 0, splatmaps);
var terrainObj = Terrain.CreateTerrainGameObject(terrainData);
```

NavMesh bake:
```csharp
var surface = terrainObj.AddComponent<NavMeshSurface>();
surface.agentTypeID = NavMesh.GetSettingsByIndex(0).agentTypeID;
// Or configure custom: radius, height, slope, step height
surface.BuildNavMesh();
```

Animator Controller:
```csharp
var controller = AnimatorController.CreateAnimatorControllerAtPath(path);
controller.AddParameter("Speed", AnimatorControllerParameterType.Float);
var rootSM = controller.layers[0].stateMachine;
var idleState = rootSM.AddState("Idle");
var runState = rootSM.AddState("Run");
var transition = idleState.AddTransition(runState);
transition.AddCondition(AnimatorConditionMode.Greater, 0.1f, "Speed");
// Blend tree:
var blendTree = rootSM.AddState("Locomotion").motion as BlendTree;
controller.CreateBlendTreeInController("Locomotion", out blendTree);
blendTree.blendParameter = "Speed";
blendTree.AddChild(idleClip, 0f);
blendTree.AddChild(walkClip, 0.5f);
blendTree.AddChild(runClip, 1f);
```

```csharp
var importer = AssetImporter.GetAtPath(fbxPath) as ModelImporter;
```

    **scene_templates.py** -- 7 generator functions:

    1. `generate_terrain_setup_script(heightmap_path: str, size: tuple[float,float,float] = (1000,600,1000), resolution: int = 513, splatmap_layers: list[dict] | None = None) -> str`
       - Creates TerrainData with configurable resolution and size
       - Reads RAW heightmap from path, sets heights
       - Configures splatmap layers if provided (each layer: texture_path, tiling)
       - Creates Terrain GameObject, saves as prefab
       - Writes result JSON with terrain object name and stats

    2. `generate_object_scatter_script(prefab_paths: list[str], density: float = 0.5, min_slope: float = 0.0, max_slope: float = 45.0, min_altitude: float = 0.0, max_altitude: float = 1000.0, seed: int = 42) -> str`
       - Uses Poisson disk sampling or grid with jitter for positions
       - Filters by terrain slope (GetInterpolatedNormal) and altitude (SampleHeight)
       - Instantiates prefabs with random rotation and scale variation
       - Groups scattered objects under a parent GameObject for organization

    3. `generate_lighting_setup_script(sun_color: list[float] = None, sun_intensity: float = 1.0, ambient_color: list[float] = None, fog_enabled: bool = True, fog_color: list[float] = None, fog_density: float = 0.01, skybox_material: str = "", time_of_day: str = "noon") -> str`
       - Creates/configures directional light (position, rotation based on time_of_day)
       - Sets RenderSettings: ambientLight, fog, fogColor, fogDensity, skybox
       - Creates Volume with dark fantasy post-processing preset (bloom, vignette, color grading)
       - Time_of_day presets: dawn, noon, dusk, night, overcast (each has sun angle + color temp)

    4. `generate_navmesh_bake_script(agent_radius: float = 0.5, agent_height: float = 2.0, max_slope: float = 45.0, step_height: float = 0.4, nav_links: list[dict] | None = None) -> str`
       - Adds NavMeshSurface component to active terrain/scene root
       - Configures agent settings
       - Calls BuildNavMesh()
       - Creates NavMeshLinks between specified start/end positions if provided

    5. `generate_animator_controller_script(name: str, states: list[dict], transitions: list[dict], parameters: list[dict], blend_trees: list[dict] | None = None) -> str`
       - States: [{name, motion_path, position}]
       - Transitions: [{from_state, to_state, conditions: [{param, mode, threshold}], has_exit_time}]
       - Parameters: [{name, type: "float|int|bool|trigger"}]
       - Blend trees: [{name, blend_param, children: [{motion_path, threshold}]}]
       - Creates AnimatorController at Assets/Animations/{name}.controller

    6. `generate_avatar_config_script(fbx_path: str, animation_type: str = "Humanoid", bone_mapping: dict | None = None) -> str`
       - Sets ModelImporter.animationType to Human or Generic
       - If Humanoid and bone_mapping provided, configures HumanDescription.human bone mappings
       - Calls SaveAndReimport()

    7. `generate_animation_rigging_script(rig_name: str, constraints: list[dict]) -> str`
       - Constraints: [{type: "two_bone_ik|multi_aim", target_path, root_path, mid_path, tip_path, source_paths, weight}]
       - Adds RigBuilder and Rig components to root
       - Adds constraint components (TwoBoneIKConstraint, MultiAimConstraint) with full configuration
       - Requires `using UnityEngine.Animations.Rigging;`

    - Parameter values appear correctly in generated code (paths, numbers, names)
    - Invalid inputs (e.g., empty states list for animator) raise ValueError
    - All templates have MenuItem attributes and write result JSON
