Create `settings_templates.py` following the exact pattern of existing template files:

Create these 11 generator functions, each returning a complete C# editor script string:

1. `generate_physics_settings_script(collision_matrix: dict[str, list[str]] | None = None, gravity: list[float] | None = None) -> str`
   - collision_matrix: dict mapping layer name to list of layers it should collide with. The script iterates all 32x32 layer pairs and calls Physics.IgnoreLayerCollision(a, b, true) for pairs NOT in the matrix
   - Uses LayerMask.NameToLayer to resolve layer names to indices
   - Optionally sets Physics.gravity if gravity provided
   - MenuItem: `VeilBreakers/Settings/Configure Physics`
   - Result JSON includes configured_layers count

2. `generate_physics_material_script(name: str, friction: float = 0.5, bounciness: float = 0.0, friction_combine: str = "Average", bounce_combine: str = "Average") -> str`
   - Creates new PhysicMaterial with specified properties
   - friction_combine / bounce_combine: Average, Minimum, Multiply, Maximum (maps to PhysicMaterialCombine enum)
   - Saves via AssetDatabase.CreateAsset to `Assets/Physics Materials/{name}.physicMaterial`
   - MenuItem: `VeilBreakers/Settings/Create Physics Material`

3. `generate_player_settings_script(company: str = "", product: str = "", color_space: str = "", scripting_backend: str = "", api_level: str = "", icon_path: str = "", splash_path: str = "", default_screen_width: int = 0, default_screen_height: int = 0) -> str`
   - Sets only non-empty parameters (skip empty strings, zero ints)
   - company -> PlayerSettings.companyName
   - product -> PlayerSettings.productName
   - color_space -> PlayerSettings.colorSpace = ColorSpace.{Linear|Gamma}
   - scripting_backend -> PlayerSettings.SetScriptingBackend(BuildTargetGroup.Standalone, ScriptingImplementation.{IL2CPP|Mono2x})
   - api_level -> PlayerSettings.SetApiCompatibilityLevel(BuildTargetGroup.Standalone, ApiCompatibilityLevel.{NET_Standard|NET_Unity_4_8})
   - icon_path -> PlayerSettings.SetIconsForTargetGroup(BuildTargetGroup.Unknown, new[] { AssetDatabase.LoadAssetAtPath<Texture2D>(path) })
   - splash_path -> PlayerSettings.SplashScreen.logos (UnityEditor.PlayerSettings.SplashScreen)
   - MenuItem: `VeilBreakers/Settings/Configure Player Settings`

4. `generate_build_settings_script(scenes: list[str] | None = None, platform: str = "", defines: list[str] | None = None) -> str`
   - scenes: list of scene asset paths -> EditorBuildSettings.scenes = new EditorBuildSettingsScene[] { new EditorBuildSettingsScene(path, true), ... }
   - platform: switch active build target via EditorUserBuildSettings.SwitchActiveBuildTarget(BuildTargetGroup.{group}, BuildTarget.{target})
   - defines: PlayerSettings.SetScriptingDefineSymbolsForGroup(BuildTargetGroup.Standalone, "DEFINE1;DEFINE2")
   - Platform mapping: StandaloneWindows64 -> (Standalone, StandaloneWindows64), Android -> (Android, Android), iOS -> (iOS, iPhone), WebGL -> (WebGL, WebGL)
   - MenuItem: `VeilBreakers/Settings/Configure Build Settings`

5. `generate_quality_settings_script(levels: list[dict]) -> str`
   - Each level dict: {name: str, shadow_distance: float, texture_quality: int (0=Full, 1=Half, 2=Quarter, 3=Eighth), anti_aliasing: int (0/2/4/8), vsync: int (0/1/2), lod_bias: float, shadow_resolution: str ("Low"/"Medium"/"High"/"VeryHigh")}
   - Uses SerializedObject on AssetDatabase.LoadMainAssetAtPath("ProjectSettings/QualitySettings.asset") to configure quality level arrays
   - MenuItem: `VeilBreakers/Settings/Configure Quality Settings`

6. `generate_package_install_script(package_id: str, version: str = "", source: str = "upm", registry_url: str = "", scopes: list[str] | None = None) -> str`
   - source="upm": generates C# using `UnityEditor.PackageManager.Client.Add("{package_id}@{version}")`
   - source="git": generates C# using `Client.Add("{package_id}")` (package_id is the full git URL)
   - source="openupm": generates C# that reads manifest.json, adds scopedRegistries entry with registry_url and scopes, then adds package to dependencies
   - All sources: wait for async operation completion via `while (!request.IsCompleted) { ... }`
   - MenuItem: `VeilBreakers/Settings/Install Package`

7. `generate_package_remove_script(package_id: str) -> str`
   - Uses `Client.Remove(package_id)`
   - MenuItem: `VeilBreakers/Settings/Remove Package`

8. `generate_tag_layer_script(tags: list[str] | None = None, layers: list[str] | None = None, sorting_layers: list[str] | None = None) -> str`
   - Tags: SerializedObject on "ProjectSettings/TagManager.asset", FindProperty("tags"), check for existing before adding, InsertArrayElementAtIndex
   - Layers: FindProperty("layers"), iterate indices 8-31 (0-7 reserved), find first empty slot for each new layer. Warn in result JSON if slots exhausted (only 24 user slots)
   - Sorting Layers: FindProperty("m_SortingLayers"), add with uniqueID and name
   - MenuItem: `VeilBreakers/Settings/Manage Tags & Layers`

9. `generate_tag_layer_sync_script(constants_cs_path: str) -> str`
   - Reads Constants.cs file content via File.ReadAllText
   - Regex matches: `public const string TAG_\w+ = "(\w+)"` and `public static readonly int LAYER_\w+ = (\d+)` (or similar patterns from VeilBreakers Constants.cs)
   - For each found tag: add to TagManager if not exists
   - For each found layer: set TagManager layers array at specified index
   - Reports drift: any tags/layers in Constants.cs not in TagManager, any in TagManager not in Constants.cs
   - MenuItem: `VeilBreakers/Settings/Sync Tags & Layers from Code`

10. `generate_time_settings_script(fixed_timestep: float = 0.02, maximum_timestep: float = 0.1, time_scale: float = 1.0) -> str`
    - Uses SerializedObject on "ProjectSettings/TimeManager.asset" for fixed_timestep and maximum_timestep
    - Or direct: Time.fixedDeltaTime = {fixed_timestep}, Time.maximumDeltaTime = {maximum_timestep}, Time.timeScale = {time_scale}
    - MenuItem: `VeilBreakers/Settings/Configure Time Settings`

11. `generate_graphics_settings_script(render_pipeline_path: str = "", fog_mode: str = "", fog_color: list[float] | None = None, fog_density: float = 0.0) -> str`
    - render_pipeline_path: loads RenderPipelineAsset and assigns to GraphicsSettings.defaultRenderPipeline (or QualitySettings.renderPipeline for per-quality)
    - fog_mode: None/Linear/Exponential/ExponentialSquared -> RenderSettings.fogMode
    - MenuItem: `VeilBreakers/Settings/Configure Graphics Settings`
