    4. `generate_multi_platform_build_script(platforms=None, development=False, namespace="") -> str` (BUILD-01):
       - Default platforms when None: list of 6 dicts:
         ```python
         [
             {"name": "Windows", "target": "StandaloneWindows64", "group": "Standalone", "backend": "IL2CPP", "extension": ".exe"},
             {"name": "Mac", "target": "StandaloneOSX", "group": "Standalone", "backend": "IL2CPP", "extension": ""},
             {"name": "Linux", "target": "StandaloneLinux64", "group": "Standalone", "backend": "IL2CPP", "extension": ""},
             {"name": "Android", "target": "Android", "group": "Android", "backend": "IL2CPP", "extension": ".apk"},
             {"name": "iOS", "target": "iOS", "group": "iOS", "backend": "IL2CPP", "extension": ""},
             {"name": "WebGL", "target": "WebGL", "group": "WebGL", "backend": "Mono2x", "extension": ""},
         ]
         ```
       - Generate C# with [MenuItem("VeilBreakers/Build/Multi-Platform Build")]
       - Use `EditorUserBuildSettings.SwitchActiveBuildTarget(BuildTargetGroup.{group}, BuildTarget.{target})` per platform
       - Set `PlayerSettings.SetScriptingBackend(NamedBuildTarget.FromBuildTargetGroup(BuildTargetGroup.{group}), ScriptingBackend.{backend})` per platform
       - Build with `BuildPipeline.BuildPlayer(options)` where locationPathName = `"Builds/" + platformName + "/Game" + extension`
       - Check `report.summary.result == BuildResult.Succeeded` before accessing report data
       - Collect per-platform results: name, result, totalSize, totalTime
       - Write JSON summary to `Temp/vb_build_results.json` with File.WriteAllText
       - `BuildOptions.Development` flag when development=True
       - Required usings: UnityEditor, UnityEditor.Build.Reporting, UnityEngine, System.IO, System.Collections.Generic, System.Text
       - Use line-based concatenation (NOT f-strings for C# body)

    5. `generate_addressables_config_script(groups=None, build_remote=False, namespace="") -> str` (BUILD-02):
       - Default groups when None: [{"name": "Default", "packing": "PackSeparately", "local": True}]
       - Each group dict: name (str), packing ("PackTogether"/"PackSeparately"/"PackTogetherByLabel"), local (bool)
       - Generate C# with [MenuItem("VeilBreakers/Build/Configure Addressables")]
       - Get or create AddressableAssetSettings: `AddressableAssetSettingsDefaultObject.Settings` ?? `AddressableAssetSettings.Create(...)`
       - For each group: `settings.CreateGroup(groupName, false, false, true, null, typeof(BundledAssetGroupSchema), typeof(ContentUpdateGroupSchema))`
       - Configure BundledAssetGroupSchema: BuildPath/LoadPath via SetVariableByName (kLocalBuildPath/kLocalLoadPath or kRemoteBuildPath/kRemoteLoadPath based on local flag)
       - Set BundleMode from packing string
       - Optionally call AddressableAssetSettings.BuildPlayerContent() if build_remote
       - Write result JSON to Temp/vb_result.json
       - Required usings: UnityEditor, UnityEditor.AddressableAssets, UnityEditor.AddressableAssets.Settings, UnityEditor.AddressableAssets.Settings.GroupSchemas, UnityEngine, System.IO, System.Collections.Generic

    6. `generate_platform_config_script(platform="android", permissions=None, features=None, plist_entries=None, webgl_memory_mb=256, namespace="") -> str` (BUILD-05):
       - When platform == "android": Generate AndroidManifest.xml content as a raw string. Write to Assets/Plugins/Android/AndroidManifest.xml using File.WriteAllText. Default permissions: ["android.permission.INTERNET"]. Default features: ["android.hardware.touchscreen"]. Use tools:node="merge" on application element. Include MAIN/LAUNCHER intent filter. Write result JSON.
       - When platform == "ios": Generate [PostProcessBuild] C# callback. PlistDocument to modify Info.plist entries (default: ITSAppUsesNonExemptEncryption=false, UIRequiresFullScreen=true). PBXProject for capabilities. Custom plist_entries as list of {"key": str, "value": str, "type": "string"|"boolean"} dicts.
       - When platform == "webgl": Generate C# editor script that sets PlayerSettings.WebGL properties: memorySize, compressionFormat (Brotli), linkerTarget, template. Write result JSON.
       - [MenuItem] under "VeilBreakers/Build/Configure {Platform}"]
       - Required usings vary by platform. Android: UnityEditor, UnityEngine, System.IO. iOS: UnityEditor, UnityEditor.Callbacks, UnityEditor.iOS.Xcode, System.IO. WebGL: UnityEditor, UnityEngine, System.IO.

    7. `generate_shader_stripping_script(keywords_to_strip=None, log_stripping=True, namespace="") -> str` (SHDR-03):
       - Default keywords_to_strip: ["DEBUG", "_EDITOR"]
       - Generate C# class implementing IPreprocessShaders
       - `callbackOrder` property returns 0
       - `OnProcessShader(Shader shader, ShaderSnippetData snippet, IList<ShaderCompilerData> data)`:
         - Build ShaderKeyword array from keywords_to_strip list
         - Iterate data backwards (i = data.Count - 1 downto 0)
         - For each keyword, check `data[i].shaderKeywordSet.IsEnabled(keyword)`, RemoveAt(i) if enabled
         - If log_stripping: Debug.Log stripped count vs total
       - Write stripping summary to Temp/vb_shader_strip_results.json via IPostprocessBuildWithReport (stripped counts per shader)
       - Required usings: UnityEditor.Build, UnityEditor.Rendering, UnityEngine, UnityEngine.Rendering, System.Collections.Generic

    8. Pure-logic validation helpers:
       - `_validate_platforms(platforms)`: Ensure each dict has name, target, group, backend keys
       - `_validate_addressable_groups(groups)`: Ensure each dict has name key, packing is valid enum string
