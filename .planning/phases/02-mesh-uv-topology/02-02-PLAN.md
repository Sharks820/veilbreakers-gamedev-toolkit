    - Test: _polygon_area_2d for a triangle with known vertices returns correct area
    - Test: _polygon_area_2d for less than 3 coords returns 0.0
    - Test: texel density formula: sqrt(uv_area / face_3d_area) * texture_size produces expected value
    - Test: A synthetic UV face with uv_area=0.01 and face_3d_area=1.0 at texture_size=1024 gives TD~102.4
  </behavior>
  <action>
    Create `Tools/mcp-toolkit/blender_addon/handlers/uv.py` with 9 handler functions:

    **1. handle_analyze_uv(params: dict) -> dict** (UV-01)
    - Accept `object_name`, optional `texture_size` (default 1024)
    - Create bmesh, get active UV layer. If no UV layer, return `{has_uvs: False, error: "No UV layer found"}`
    - Per-face metrics: 3D area via `face.calc_area()`, UV area via `_polygon_area_2d()` (shoelace formula on `loop[uv_layer].uv` coords)
    - Stretch: ratio of UV area to 3D area per face. Compute median, then average deviation from median.
    - Texel density per face: `sqrt(uv_area / face_3d_area) * texture_size`. Compute avg/min/max/variance_ratio.
    - Island counting: `_count_uv_islands(bm, uv_layer)` using BFS flood fill. Seams break connectivity. Check UV-edge sharing via `_faces_share_uv_edge()`.
    - Overlap detection: `_count_uv_overlaps(bm, uv_layer)` using spatial grid hash (2D grid at 1/100 resolution). For meshes >10k faces, sample 10% and extrapolate. Flag as estimate if sampled.
    - Seam count: `sum(1 for e in bm.edges if e.seam)`
    - Return structured dict with all metrics per research spec.

    **2. handle_unwrap_xatlas(params: dict) -> dict** (UV-02)
    - Accept `object_name`, optional `max_chart_area`, `normal_deviation_weight`, `max_iterations`, `padding` (default 2), `resolution` (default 1024), `bilinear` (default True), `rotate_charts` (default True)
    - Try `import xatlas`. On ImportError, raise with install instructions.
    - Extract mesh data via numpy: vertices (Nx3 float32), normals (Nx3 float32), triangulated face indices (Mx3 uint32). Fan-triangulate n-gons.
    - Build tri-to-original-face mapping during triangulation for UV write-back.
    - Create xatlas.Atlas(), add_mesh, configure ChartOptions and PackOptions from params, generate().
    - Get vmapping, indices, uvs from atlas[0].
    - Write UVs back to bmesh: create/get UV layer, map xatlas UV coords back to original mesh loops via vmapping. Handle seam vertices (multiple UVs per vertex) by using the tri-to-face mapping.
    - `bm.to_mesh(mesh); mesh.update(); bm.free()`
    - Return dict: `{object_name, atlas_width, atlas_height, chart_count, uv_layer: "UVMap"}`

    **3. handle_unwrap_blender(params: dict) -> dict** (UV-02 fallback)
    - Accept `object_name`, optional `method` ("smart_project" or "angle_based", default "smart_project"), optional `angle_limit` (default 66 degrees)
    - Use context override to enter edit mode, select all, run `bpy.ops.uv.smart_project()` or `bpy.ops.uv.unwrap()`, return to object mode
    - Return dict with basic metrics

    **4. handle_pack_islands(params: dict) -> dict** (UV-03)
    - Accept `object_name`, optional `margin` (default 0.001)
    - Enter edit mode with context override, select all, run `bpy.ops.uv.pack_islands(margin=margin)`, return to object mode
    - Return success dict

    **5. handle_generate_lightmap_uv(params: dict) -> dict** (UV-04)
    - Accept `object_name`, optional `padding` (default 4), optional `resolution` (default 1024)
    - Create second UV layer named "UV2" via bmesh: `bm.loops.layers.uv.new("UV2")`
    - Set UV2 as active layer
    - Run xatlas unwrap targeting UV2 with lightmap-specific options: higher padding (4), bilinear=True, no chart area limit
    - Write results to UV2 layer only (preserve UV1)
    - Return dict: `{object_name, uv_layer: "UV2", atlas_width, atlas_height}`

    **6. handle_equalize_density(params: dict) -> dict** (UV-05)
    - Accept `object_name`, optional `target_density` (None = use median), optional `texture_size` (default 1024)
    - Calculate per-island texel density
    - Determine target (median or user-specified)
    - Scale each island's UVs around island center by `target_density / island_density`
    - Repack islands via bpy.ops.uv.pack_islands
    - Return before/after density stats

    **7. handle_export_uv_layout(params: dict) -> dict** (visual verification)
    - Accept `object_name`, optional `size` (default 1024), optional `opacity` (default 0.25)
    - Enter edit mode, select all faces, run `bpy.ops.uv.export_layout()` to temp PNG file
    - Return `{filepath, size, format: "png"}`
    - If UV export operator fails (no UV editor), fall back to manual Pillow-based UV line rendering from bmesh UV data

    **8. handle_set_active_uv_layer(params: dict) -> dict** (utility)
    - Accept `object_name`, `layer_name`
    - Set the named UV layer as active on the mesh
    - Return confirmation

    **9. handle_ensure_xatlas(params: dict) -> dict** (setup)
    - Try `import xatlas`. If succeeds, return `{installed: True, version}`.
    - If fails, run `subprocess.check_call([sys.executable, '-m', 'pip', 'install', 'xatlas==0.0.11'])`
    - Re-import and confirm
    - Return `{installed: True, version, just_installed: True}`

    **Helper functions (module-level, not handlers):**
    - `_polygon_area_2d(coords: list) -> float` -- Shoelace formula
    - `_count_uv_islands(bm, uv_layer) -> int` -- BFS flood fill
    - `_faces_share_uv_edge(f1, f2, edge, uv_layer) -> bool` -- UV connectivity check
    - `_count_uv_overlaps(bm, uv_layer) -> int` -- Grid-hash overlap detection
    - `_get_island_faces(bm, uv_layer) -> list[list[BMFace]]` -- Return list of face groups per island

    **Tests** (in test_uv_handlers.py):
    - Unit test `_polygon_area_2d` with known polygons (unit square, triangle, pentagon)
    - Unit test texel density formula with known inputs
    - Tests should NOT require bpy -- test pure math functions by importing them directly
    - Mark any Blender-dependent tests with `@pytest.mark.blender`

    **1. Add blender_uv compound tool to blender_server.py:**

    Add a new `@mcp.tool()` function `blender_uv` after the `blender_mesh` tool (or after `blender_export` if Plan 02-01 hasn't run yet):

    ```python
    @mcp.tool()
    async def blender_uv(
        action: Literal[
            "analyze", "unwrap", "unwrap_blender", "pack",
            "lightmap", "equalize", "export_layout", "set_layer", "ensure_xatlas"
        ],
        object_name: str | None = None,
        texture_size: int = 1024,
        padding: int = 2,
        resolution: int = 1024,
        margin: float = 0.001,
        layer_name: str | None = None,
        method: str = "smart_project",
        max_chart_area: float | None = None,
        normal_deviation_weight: float | None = None,
        max_iterations: int | None = None,
        rotate_charts: bool = True,
        target_density: float | None = None,
        size: int = 1024,
        opacity: float = 0.25,
        capture_viewport: bool = True,
    ) -> list[str | Image]:
        """UV mapping analysis, unwrapping, packing, and optimization.

        Actions:
        - analyze: UV quality analysis (stretch, overlap, island count, texel density, seams)
        - unwrap: Automatic UV unwrap via xatlas (high quality, configurable)
        - unwrap_blender: Blender native UV unwrap (smart_project or angle_based)
        - pack: UV island packing optimization
        - lightmap: Generate lightmap UV2 for Unity (separate from UV1)
        - equalize: Texel density equalization across all UV islands
        - export_layout: Export UV layout as PNG image for visual review
        - set_layer: Set active UV layer by name
        - ensure_xatlas: Install xatlas into Blender Python if not present
        """
    ```

    Route each action to the appropriate `send_command("uv_*", params)` call:
