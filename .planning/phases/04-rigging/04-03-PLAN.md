    6. **Pure-logic validation functions:**

    `_validate_ik_params(params: dict) -> dict`:
    - Validates: bone_name is string, chain_length >= 1 and <= 20, constraint_type in ("IK", "SPLINE_IK"), pole_angle is float
    - For SPLINE_IK: requires curve_points >= 2
    - Returns: {"valid": bool, "errors": list[str]}

    `_validate_spring_params(bone_names: list[str], stiffness: float, damping: float, gravity: float) -> dict`:
    - Validates: bone_names is non-empty, stiffness in [0, 1], damping in [0, 1], gravity >= 0
    - Returns: {"valid": bool, "errors": list[str], "bone_count": int}

    `_validate_ragdoll_spec(bone_collider_map: dict) -> dict`:
    - Validates each entry: shape in ("BOX", "CAPSULE"), mass > 0, dimensions > 0, joint angles in valid range
    - Returns: {"valid": bool, "errors": list[str], "collider_count": int}

    `_validate_retarget_mapping(source_bones: list[str], target_bones: list[str], mapping: dict[str, str]) -> dict`:
    - Validates all source bones in mapping exist in source_bones list
    - Validates all target bones in mapping exist in target_bones list
    - Returns: {"valid": bool, "errors": list[str], "mapped_count": int, "unmapped_source": list, "unmapped_target": list}

    `_validate_shape_key_params(name: str, vertex_offsets: dict) -> dict`:
    - Validates name is non-empty string (no special chars except underscore/hyphen)
    - Validates vertex_offsets: keys are non-negative ints, values are 3-element tuples/lists of floats
    - Returns: {"valid": bool, "errors": list[str], "vertex_count": int}

    7. **Blender-dependent handlers:**

    `handle_setup_facial(params: dict) -> dict` (RIG-04):
    - Gets rig object by name
    - Adds FACIAL_BONES to existing armature in edit mode (parent "head" references the head bone of the rig)
    - Optionally applies a monster expression preset from MONSTER_EXPRESSIONS as a pose library entry
    - Returns: {"facial_bones_added": int, "expressions_stored": list[str], "rig_name": str}

    `handle_setup_ik(params: dict) -> dict` (RIG-05):
    - Gets rig object, bone_name, constraint params
    - Validates via _validate_ik_params
    - For standard IK: pbone.constraints.new('IK') with target, subtarget, chain_count, pole_target
    - For SPLINE_IK: creates curve object from control points, adds SPLINE_IK constraint
    - Optionally adds LIMIT_ROTATION constraints for joint limits
    - Returns: {"constraint": str, "chain_length": int, "type": "IK"|"SPLINE_IK"}

    `handle_setup_spring_bones(params: dict) -> dict` (RIG-06):
    - Gets rig object, bone_names list, stiffness, damping, gravity
    - Validates via _validate_spring_params
    - For each bone: adds DAMPED_TRACK constraint (influence=stiffness), COPY_ROTATION from parent (influence=damping * decay)
    - Stores spring params as custom properties on each pose bone
    - Returns: {"spring_bones": list[str], "stiffness": float, "damping": float, "gravity": float}

    `handle_setup_ragdoll(params: dict) -> dict` (RIG-11):
    - Gets rig object, bone_collider_map (or preset name from RAGDOLL_PRESETS)
    - Validates via _validate_ragdoll_spec
    - For each bone: creates mesh collider (cylinder for CAPSULE, cube for BOX), positions at bone, adds rigid_body, parents to bone
    - Creates rigid body constraints between adjacent colliders with joint limits
    - Returns: {"colliders": list[str], "joint_count": int, "preset_used": str|None}

    `handle_retarget_rig(params: dict) -> dict` (RIG-12):
    - Gets source rig, target rig, bone mapping dict
    - Validates via _validate_retarget_mapping
    - For each mapped pair: adds COPY_ROTATION + COPY_LOCATION constraints on target bones pointing to source bones
    - Influence scaling based on bone length ratio for size differences
    - Returns: {"mapped_bones": int, "source_rig": str, "target_rig": str, "unmapped": list}

    `handle_add_shape_keys(params: dict) -> dict` (RIG-13):
    - Gets mesh object, shape_key_name, mode from params
    - mode="expression": creates shape key from MONSTER_EXPRESSIONS facial bone poses converted to vertex offsets
    - mode="damage": creates damage shape key (random vertex displacement scaled by convexity for realistic damage)
    - mode="custom": creates shape key from explicit vertex_offsets dict
    - Ensures Basis shape key exists first
    - Returns: {"shape_key": str, "vertices_modified": int, "total_shape_keys": int}
