### Plan 2.2: Enforce Seed-Based RNG (MESH-01.1a)

**Goal:** Ensure all 284 generators use `random.Random(seed)` instead of global state.

**Actions:**
1. Add RNG audit function:
   ```python
   def check_rng_usage(filepath):
       """Scan for random.random() usage without seed parameter."""
       with open(filepath) as f:
           content = f.read()
       
       # BAD: Using global random
       if 'import random' in content and 'random.random()' not in content:
           return {'status': 'FAIL', 'line': line}
       
       # BAD: Using random without seed
       if re.search(r'random\.random\(\)', content):
           return {'status': 'FAIL', 'line': line}
       
       # GOOD: Using seeded RNG
       if re.search(r'random\.Random\(.*seed\)', content):
           return {'status': 'PASS', 'line': line}
       
       return {'status': 'UNKNOWN', 'message': 'No random usage found'}
   ```
2. Audit `procedural_meshes.py` for RNG compliance
3. Fix violations: Change `import random` + `random.random()` to `import random as rnd; rng = random.Random(seed)` pattern
4. Ensure all generator functions accept `seed` parameter

**Success Criteria:**
- RNG audit script created
- All 284 generators scanned
- Non-compliant generators identified
- Compliance fixes implemented
